easypwgen

A simple, secure password generator (made by Hayden Schiff)

This is an extremely simple password generator inspired the comic xkcd #936: Password Strength. It will generate a random password for you by selecting a few random words from the dictionary. This page is deliberately barebones and simple, so that any developer can easily peek at the source code and verify that this website is not doing anything evil.

To generate a random password, just click the "Go!" button below. If you don't like the password you get, just keep clicking until you find one that you like.

Your password will appear here once you click the "Go!" button.
